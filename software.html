Free EE software

Name Platforms Description
LTSpice Windows, macOS Free SPICE software provided by Analog Devices. Barebones UI, but that is to be expected from ECAD.
Qucs-S Windows, macOS, Linux, FreeBSD Continuation of Qucs. Uses 4 different SPICE backends including XYCE. Plethora of simulation options
QucsStudio Windows Alternative to Qucs-S. Closed source and updated infrequently, however the main SPICE engine it uses is phenominal for RF especially Harmonic Balance
Micro-cap Windows Formerly propriety software. Made free with end of life in 2021. Plethora of examples and lots of functionality. Dated UI
Digital Java Not a very searchable name. By Hneemann on GitHub. Excellent digital simulation software with support for HDL via GHDL and IcarusVerilog
GHDL Windows, macOS, Linux, FreeBSD Free and open source VHDL simulator. Can be used in conjunction with yosys and icestorm to synthesize VHDL code to FPGAs. Supports most of VHDL-2008 as of October 24 2024
Icarus Verilog Windows, macOS, Linux, FreeBSD Free and open source Verilog simulator. Supports to the 2012 standard which supports SystemVerilog
Verilator Linux, FreeBSD Industry grade verilog simulator. Converts Verilog to HLS code
GTKWave Windows, macOS, Linux, FreeBSD Waveform viewer for mainly verilog. However GHDL can generate a format that can be used for VHDL testbenches
yosys Windows, macOS, Linux, FreeBSD RTL synthesis tool for verilog, but plugins can make it interface with VHDL via GHDL. Backbone of the open source FPGA synthesis toolchain
nextpnr Windows, macOS, Linux, FreeBSD Place and route tool for Lattice's iCE40, ECP5, and Nexus FPGAs
icestorm Windows, macOS, Linux, FreeBSD Used to program the bitstream to iCE40 FPGAs.
KiCAD Windows, macOS, Linux, FreeBSD Excellent schematic layout and PCB designing tool. Includes features engineers would have killed someone for just a decade ago. Includes calculators for EMI and RF related uses.
Magic VLSI Linux, FreeBSD Extremely famous VLSI layout tool. Made by the inventor of the Tcl scripting language. Used in DARPA's OpenROAD project
Electric VLSI Java Another free VLSI tool. Development ceased in 2017
Octave Windows, macOS, Linux, FreeBSD Matlab clone
openEMS Windows, Linux, FreeBSD FDTD field solver. No GUI, uses either Matlab, python, or Octave as a scripting interface
